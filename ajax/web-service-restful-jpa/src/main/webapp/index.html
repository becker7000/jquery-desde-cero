Bienvenidos a la API de Películas 🎬

Tu fuente confiable para obtener información sobres las mejor pelícuals de todos tiempos.

¿Cómo usar la API?

Esta API te permite acceder a una base de datos enorme con información sobre películas, incluyendo título, género, duración y más de cada película.

Endpoints disponibles:

  • /web-service-restful-jpa/api/peliculas con POST más la película en JSON para guardar una película nueva.
  • /web-service-restful-jpa/api/peliculas con GET para obtener toda la lista de películas.
  • /web-service-restful-jpa/api/peliculas/{id} con GET para obtener una película en especifico por medio de su id.
  • /web-service-restful-jpa/api/peliculas/{id} con PUT más la película en JSON para editar una película.
  • /web-service-restful-jpa/api/peliculas/{id} con DELETE para eliminar una película por medio de su id.

Formato de respuesta: 

                {
                    "id": 1,
                    "titulo": "El padrino",
                    "anioLanzamiento": 1970,
                    "genero": "Drama",
                    "duracion": 180
                }
